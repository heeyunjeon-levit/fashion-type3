## üìù Documentation Guidelines

When adding new documentation:

1. **Choose the right folder**:
   - Feature docs ‚Üí `/features/`
   - Deployment guides ‚Üí `/deployment/`

   - Guides: `TOPIC_GUIDE.md`

3. **Include key info**:
   - Date and context
   - Problem/solution summary
   - Code examples
   - Related files

4. **Archive old docs**:
   - Move outdated docs to `/archived/`
   - Keep root clean
